# threat.txt — seed file for criminal/botnet IPs
#
# Этот файл объединяется с URL-источниками (см. lists/README.md).
# На ноде кладётся в /etc/shieldnode/lists/threat.txt.
#
# Основные источники качаются ежедневно:
#   - Spamhaus DROP list (https://www.spamhaus.org/drop/drop.txt)
#   - FireHOL Level 1 (https://iplists.firehol.org/files/firehol_level1.netset)
#
# Сюда — только high-confidence криминал (botnet C2, ransomware ops, etc).
# Низкий false-positive rate важен — это критичный set, ложный бан легитимного
# IP здесь хуже чем пропустить настоящий threat (для них есть scanner+rate-limit).
#
# Формат: один IP или CIDR на строку. Комментарии через #.

# === Examples ===
# 198.51.100.0/24    # known botnet subnet, observed 2025-12-15
