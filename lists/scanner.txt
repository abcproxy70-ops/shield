# scanner.txt — seed file for known scanner IPs
#
# Этот файл объединяется с URL-источниками (см. lists/README.md).
# На ноде кладётся в /etc/shieldnode/lists/scanner.txt.
#
# Большинство IP качаются из публичных источников каждые 6 часов:
#   - shadow-netlab/traffic-guard-lists (Shodan, Censys, общие сканеры)
#   - tread-lightly/CyberOK_Skipa_ips (российские госсканеры SKIPA)
#   - MISP/CIRCL warning lists
#
# Сюда добавляй свои findings которых нет в публичных списках.
# Формат: один IP или CIDR на строку. Комментарии через #.
