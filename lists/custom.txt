# custom.txt — твой личный blocklist (синкается с github каждые 6ч)
#
# ⚠️ ВАЖНО:
#   - Этот файл хранится в github репо и автоматически качается на все ноды
#     каждые 6 часов (https://raw.githubusercontent.com/abcproxy70-ops/shield/main/lists/custom.txt)
#   - На ноде он пишется в /etc/shieldnode/lists/custom.txt (read-only от sync)
#   - НЕ редактируй custom.txt на ноде напрямую — будет перезатёрто при следующем sync
#
# Чтобы добавить IP:
#   - Постоянно для всех нод → PR в этот файл на github
#   - Только на конкретной ноде → /etc/shieldnode/lists/custom-local.txt
#     (не синкается, локальные изменения сохраняются)
#
# Формат: один IP или CIDR на строку. Комментарии через #.

# === Твои findings ===
# Пример:
# 198.51.100.42      # observed attack on node-de-01, 2025-12-15
# 203.0.113.0/24     # bad subnet, multiple complaints
